# YongStockDesk 1.0.0

이 문서는 1.0.0 소스 범위와 릴리스 체크 기준이다. arm64/x64 설치 아티팩트는 각 아키텍처의 생성·검증 리포트가 실제로 통과한 뒤에만 제공 완료로 표시한다.

## 지원 환경

- macOS 14 이상
- 패키징 대상은 Apple Silicon `arm64`와 Intel Mac `x64`이며, 현재 제공 여부는 아키텍처별 DMG·ZIP·manifest·install verification 결과로 판정
- 앱 번들 이름은 기존 데이터 호환을 위해 `StockAnalysis.app` 유지
- Node 22.17.0과 TypeScript sidecar를 앱 안에 포함하므로 별도 Node 설치 불필요

## 주요 기능

- 첫 실행 시 분석 전용, Toss 연결, Upbit·Bithumb 연결 경로를 선택하는 온보딩
- US·KR 종목 검색, 일봉 분석, 시장 브리핑, 모의투자와 전략 시뮬레이션
- Federal Reserve, SEC, BEA 공식/RSS 뉴스의 2분 polling과 종목·거시 뉴스 필터
- 종목별 커뮤니티 공포·과열·분열 점수, 표본 신뢰도, 소스 상태와 원문 근거
- Toss token·계좌·보유·미체결 준비 점검, credential 변경 시 token cache 격리, 요청 timeout과 안전한 재시도
- Upbit·Bithumb credential, 잔고, 주문 가능 정보, 최신 REST 현재가, 최소 주문금액과 지정가 사전검증
- arm64/x64 DMG·ZIP·manifest 생성 도구, 앱·sidecar 버전 일치 검사와 설치 검증 도구

## 안전 범위

- credential 등록, 뉴스·민심, 사전검증과 시뮬레이션은 주문을 제출하지 않는다.
- Toss 실제 주문은 데스크톱 1.0.0에서 차단한다. credential·계좌·보유·미체결·주문 사전검증은 실제 주문 없이 사용할 수 있다.
- Upbit·Bithumb 실제 주문은 1.0.0에서 차단한다. 체결·부분체결·미체결·취소 동기화와 재시작 멱등성 검증이 포함된 뒤 별도 버전에서 연다.
- 뉴스와 커뮤니티 민심은 참고 근거이며 주문 입력으로 사용하지 않는다.

## 알려진 제한

- 뉴스는 WebSocket 또는 유료 실시간 피드가 아니라 공식/RSS polling이다.
- 코인 실시간 캔들 WebSocket은 아직 연결되지 않았다. 현재 차트의 `30일·90일·1년` 버튼은 일봉 표시 범위다.
- 커뮤니티 소스는 정책과 외부 설정에 따라 비활성 또는 `configuration-required`가 될 수 있다. Reddit 공식 OAuth credential은 앱의 뉴스·알림 화면에서 Keychain에 선택 저장할 수 있다.
- Developer ID 서명과 Apple notarization이 없는 ad-hoc 패키지는 다른 Mac에서 Gatekeeper 경고가 표시될 수 있다.
- 경고 없는 공개 배포 완료 판정에는 Developer ID, notarization, stapling과 실제 Apple Silicon·Intel Mac 설치 검증이 필요하다.
- arm64와 x64 DMG는 Apple Silicon 호스트에서 각각 native/Rosetta로 설치·실행 검증했다. 실제 Intel 하드웨어 검증은 아직 남아 있다.
