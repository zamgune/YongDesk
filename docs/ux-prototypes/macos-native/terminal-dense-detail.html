구현 범위: P0-P1은 v1 대상, P2는 후순위

P0/P1 기능은 위 탭 구조 안에 녹였습니다. P2는 화면 밖 보조 기능으로 두고, 핵심 거래 플로우가 안정된 뒤 붙이는 편이 낫습니다.

P0

리스크 시나리오 패널

P0

관심목록 단위 알림

P1

포지션 플레이북

P2

오디오 Squawk 모드

중요 알림만 짧은 음성으로 읽어주는 모드입니다. 화면을 계속 보지 않아도 뉴스와 보유종목 리스크를 놓치지 않게 합니다.

macOS 앱에서는 이 화면을 기본 데스크톱 워크스페이스로 두고, 메뉴바 미니뷰에는 Sidecar 상태, 긴급 알림, 모의/실거래 게이트, 긴급 중지만 노출하는 구성이 적합합니다. 모바일 대응은 보조 목표이며, 좁은 화면에서는 관심목록 → 판단 패널 → 이벤트 테이프 → 차트 순서로 접는 것이 좋습니다.
